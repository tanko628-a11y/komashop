## 📊 測定指標の定義

### **1. ウィンレート（Win Rate）**

```
定義: 利益トレード数 / 全トレード数 × 100

計算例:
  利益トレード: 55回
  損失トレード: 45回
  全トレード: 100回
  ウィンレート: 55/100 = 55%

評価:
  > 60%: 優秀
  55-60%: 良好
  50-55%: 許容
  < 50%: 改善要
```

### **2. プロフィットファクター（PF）**

```
定義: 総利益 / 総損失

計算例:
  総利益: $15,000
  総損失: $10,000
  PF: 15,000 / 10,000 = 1.5

評価:
  > 2.0: 優秀（フロック）
  1.5-2.0: 良好
  1.3-1.5: 許容
  < 1.3: 改善要
```

### **3. 最大ドローダウン（Maximum Drawdown）**

```
定義: ピークから最低底までの下落幅

計算例:
  ピーク資金: $10,000
  最低底: $8,000
  MD: (10,000-8,000)/10,000 = 20%

評価:
  < 15%: 優秀
  15-20%: 良好
  20-25%: 許容
  > 25%: リスク大
```

### **4. Sharpe比（Sharpe Ratio）**

```
定義: (平均リターン - リスクフリーレート) / リターンの標準偏差

評価:
  > 2.0: 優秀
  1.0-2.0: 良好
  0.5-1.0: 許容
  < 0.5: 改善要
```

### **5. 利益：損失比（Profit/Loss Ratio）**

```
定義: 平均利益 / 平均損失

計算例:
  平均利益: 250 pips
  平均損失: 100 pips
  P/L比: 250/100 = 2.5

評価:
  > 2.0: 優秀
  1.5-2.0: 良好
  1.0-1.5: 許容
  < 1.0: 改善要
```
