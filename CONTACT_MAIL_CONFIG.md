## 📋 メール受け取り設定手順

### **Step 1: WordPress 管理画面で Contact Form 7 を開く**

```
左サイドメニュー → Contact Form 7
既存フォーム「お問い合わせ」をクリック
```

### **Step 2: 「メール」タブを開く**

```
Tab: 「メール」（または「Mail」）
```

### **Step 3: メール設定を確認・修正**

```
【Mail タブの設定】

To（宛先）:
  kei@komasanshop.com

From（差出人名）:
  凪フィナンシャル お問い合わせ対応

Subject（件名）:
  [新規お問い合わせ] [your-subject]

Message Body（メール本文）:
  ────────────────────────────────────
  新しいお問い合わせがあります。
  
  【申込者情報】
  名前: [your-name]
  メール: [your-email]
  
  【内容】
  件名: [your-subject]
  
  メッセージ:
  [your-message]
  
  ────────────────────────────────────
  
  Exclude blank fields: チェック ✓
  Use HTML content type: チェック ✓
```

### **Step 4: 「変更を保存」をクリック**

```
下部の「保存」または「Save」ボタンをクリック
```

## 📧 自動返信メール設定（Mail 2）

### **Step 1: 「メール(2)」タブを開く**

```
Tab: 「メール(2)」（または「Mail 2」）
※ない場合は「メール」タブに追加設定
```

### **Step 2: 自動返信メール設定**

```
【Mail 2（自動返信）タブの設定】

To（送信先）:
  [your-email]  ← 顧客のメールアドレス

From（差出人名）:
  凪フィナンシャル サポートチーム

Subject（件名）:
  お問い合わせをお受けしました

Message Body（メール本文）:
  ────────────────────────────────────
  [your-name] 様へ
  
  いつもお世話になっております。
  凪フィナンシャルです。
  
  この度はお問い合わせいただき、
  ありがとうございます。
  
  下記の内容でお受けしました：
  
  【申込情報】
  件名: [your-subject]
  
  【対応予定】
  お問い合わせの内容を確認の上、
  24時間以内にご返信させていただきます。
  
  ご不明な点やご不安なことがあれば、
  遠慮なくお問い合わせください。
  
  ────────────────────────────────────
  凪フィナンシャル
  サポートチーム
  
  Use HTML content type: チェック ✓
```

### **Step 3: 「変更を保存」をクリック**

```
下部の「保存」または「Save」ボタンをクリック
```


2. **テストデータを入力**
   ```
   名前: テスト 太郎
   メール: test@example.com
   件名: テスト お問い合わせ
   メッセージ: これはテストメールです
   ```

3. **送信ボタンをクリック**

4. **メール受信確認**
   ```
   ✓ kei@komasanshop.com に受信メールが届く
   ✓ test@example.com に自動返信メールが届く
   ```

## ⚠️ トラブルシューティング

### **メールが届かない場合**

```
原因1: To アドレスが誤っている
解決: kei@komasanshop.com に修正

原因2: メール設定が保存されていない
解決: 「変更を保存」をもう一度クリック

原因3: スパムフォルダに入っている
解決: スパムフォルダを確認、
      kei@komasanshop.com をホワイトリストに追加

原因4: Contact Form 7 が有効化されていない
解決: プラグイン → Contact Form 7 を有効化確認
```

### **自動返信メールが送信されない場合**

```
原因: Mail(2) タブが設定されていない
解決: Mail(2) を新規作成して設定
```
