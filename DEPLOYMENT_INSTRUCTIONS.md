## ⚠️ 問題が発生した場合

### **フッターが表示されない**

```
1. footer.php 修正を確認
   - コード追加位置正しいか？
   - HTML 構文エラーなし？
2. ブラウザキャッシュをクリア
3. WordPress キャッシュプラグインがあれば キャッシュ削除
```

### **Contact Form 7 が表示されない**

```
1. プラグインが有効化されているか確認
2. お問い合わせページが存在するか確認
3. ショートコード [contact-form-7] が設定されているか確認
4. ブラウザキャッシュをクリア
```

### **メールが届かない**

```
1. To アドレスが kei@komasanshop.com か確認
2. Contact Form 7 メール設定を再度確認
3. WordPress メール送信機能確認
4. スパムフォルダを確認
5. ホスティング会社に メール送信機能確認
```
