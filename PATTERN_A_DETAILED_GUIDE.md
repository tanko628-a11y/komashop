#### 金銭面の優位性（合法性）

```
【なぜこの形式なのか】

❌ シグナル配信: 「買い」「売り」を指示
   → 金融商品取引法の「投資助言」
   → 登録なしでは違法

✅ 分析レポート: 「過去データ分析」を情報提供
   → 「情報販売」「コンテンツ販売」
   → 合法（免責事項明記で安全）

【法的根拠】
- 「過去データの分析」= 情報
- 「将来の予測」= 助言扱いになる可能性
  → そのため「参考値」と明記

結論: 「何が起こったか」は説明OK
    「何をすべきか」は指示できない
```
