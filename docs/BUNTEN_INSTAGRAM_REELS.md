# 🎬 Instagram Reels - ブンテン総菜部「日曜推奨品紹介」

**最終更新**: 2026-04-23  
**プロジェクト**: フードセンターブンテン 総菜部 動画マーケティング  
**ステータス**: ✅ 運用中（自動化完成）

---

## 📋 プロジェクト概要

### 目的

フードセンターブンテン 総菜部の「日曜日のお勧め品」を動画（Instagram Reels）で紹介し、週末の来店客を増加させる。

### 対象

- **ターゲット**: フードセンターブンテン顧客（函館地域の来店客）
- **効果**: 週末の売上促進
- **投稿頻度**: 毎週日曜朝 6:00

---

## 🎥 コンテンツ設計

### ビデオフォーマット

```
動画長：30～60 秒
解像度：1080×1920（Instagram Reels 推奨）
フレームレート：30fps
BGM：季節・シーン別
```

### コンテンツ構成

```
【30～60秒のビデオ構成】

┌─────────────────────────────┐
│ オープニング（5秒）          │
│ 「今週のお勧め🍣」           │
├─────────────────────────────┤
│ メイン商品（40～45秒）       │
│ 3～5 品目を紹介              │
│ ├─ 商品名                   │
│ ├─ 推しポイント             │
│ │  （美味しさ・新鮮さ・    │
│ │   期間限定など）          │
│ └─ 価格帯                   │
├─────────────────────────────┤
│ クロージング（5秒）          │
│ 「日曜は総菜部へ！」        │
└─────────────────────────────┘
```

### CTA（Call to Action）

- 来店促進
- SNS フォロー促進

## 🔧 制作パイプライン

### 現在のワークフロー

```
【撮影】
土曜夕方 30分
スマートフォン撮影
    ↓
【データ転送】
Cloudflare Tunnel → Node.js → FastAPI
    ↓
【自動編集】
FFmpeg で処理
  ├─ Ken Burns ズーム効果
  ├─ drawtext で商品名・価格オーバーレイ
  ├─ BGM 自動挿入
    ↓
【受け取り】
Gmail で PC から動画受け取り
    ↓
【投稿】
日曜朝 6:00 に手動で Instagram 投稿
```

### 技術スタック

- **撮影**: スマートフォン
- **転送**: Cloudflare Tunnel
- **処理**: Node.js v24.13.1 + FastAPI
- **編集**: FFmpeg
- **メディア受信**: Gmail
- **投稿**: Instagram 公式アプリ

## ⚠️ 現在の課題

### BGM バリエーション

**課題**: マンネリ化  
**原因**: 同じ BGM ばかり使用

**改善案**:
```
□ BGM 自動選択の多様化
  - 季節感を反映
  - 商品カテゴリ別
    * 揚げ物 → アップテンポ
    * 刺し身 → 落ち着き系
    * サラダ → 爽やか系
```

### ビジュアルエフェクト

**課題**: パターン化  
**原因**: Ken Burns ズームのみ使用

**改善案**:
```
□ エフェクト拡張
  - Ken Burns 以外のトランジション
    * フェード
    * スライド
    * ズーム In/Out
  - テキストアニメーション
    * フェード
    * スライド
    * タイプライター効果
```

### Analytics 未活用

**課題**: パフォーマンス分析がない  
**原因**: Insights を確認していない

**改善案**:
```
□ Analytics 連携
  - Instagram Insights で週間パフォーマンス確認
  - 指標記録
    * いいね数
    * リーチ
    * 保存数
    * シェア数
  - 週間レビュー
```
